# 接口列表

## 1. 首页
- 接口名: Index/index
- 请求参数
```
{
    period: 年级 0: 小学，1: 初中
}
```
- 返回参数
```
{

    msg: 消息
    data: {
        slide_list:[
            {
                id
                title: 幻灯片标题
                img: 幻灯片图片
                type: 动作类型: 0，0: 链接地址，1: 文本内容
                type_value: 动作值
                is_del: 是否删除
                sort: 排序
            }
        ],
        count: {
            teacher_count:
            student_count:
        }
        
        redian:[
            {
                id
                title: 标题
                img: 图片
                url: 视频或音频
                keywords: 关键词，多个用逗号隔开
                type_id: 所属分类ID
                period: 0: 小学，1: 初中
                flag: 推荐: 1，热门: 2
                author: 作者
                pv_num: PV数量
                ip_num: IP数量
                sort: 排序字段
                add_time: 添加时间
                add_date: 添加日期（20170904）
            },
            ...
        ],
        cihui:[
            同上
        ],
        yufa:[
            同上
        ],
        juxing:[
            同上
        ],
        zuowen:[
            同上
        ],
        tingli:[
            同上
        ],
        weike:[
            同上
        ],
        shequ:[
            {
                id: 话题ID
                user_id: 发表人
                user_name: 昵称
                face: 用户头像
                content: 话题内容
                images: [
                    'http://xxx',
                    'http://xxx'
                ],

            ...

}
```

## 2. 新闻详情
- 接口名: News/info
- 接口参数
```
{
    id: 新闻ID
}
```
- 返回参数
```
{
    code: 0|1
    msg: 消息
    data: {
        info:{
            id
            title: 标题
            img: 图片
            url: 视频或音频
            url_type: 1: 音频 2: 视频
            keywords: 关键词，多个用逗号隔开
            type_id: 所属分类ID
            period: 0: 小学，1: 初中
            flag: 推荐: 1，热门: 2
            author: 作者
            pv_num: PV数量
            ip_num: IP数量
            sort: 排序字段
            add_time: 添加时间
            add_date: 添加日期（20170904）
            body: 新闻详情
        },
        recommend:[
            {
                id
                title: 标题
                img: 图片
                url: 视频或音频
                keywords: 关键词，多个用逗号隔开
                type_id: 所属分类ID
                period: 0: 小学，1: 初中
                flag: 推荐: 1，热门: 2
                author: 作者
                pv_num: PV数量
                ip_num: IP数量
                sort: 排序字段
                add_time: 添加时间
                add_date: 添加日期（20170904）
            }
            ...
        ]
    }
}
```


## 3. 回复
- 接口名: News/follow
- 请求参数
```
{
    user_id: 回复人
    news_id: 新闻ID
    follow_id: 0
    content: 回复内容
    notice: {} //
}
```
- 返回参数
```
{
    code: 0|1
    msg: 消息
    data: {
        id: 
        user_id: 回复人
        news_id: 新闻ID
        follow_id: 回复ID
        notice: {}
        status: 状态
        add_time: 回复时间
    }
}
```

## 4. 新闻回复列表
- 接口名: News/followlist
- 请求参数
```
{
    news_id: 新闻ID
    page: 页码
    limit: 每页数量
}
```
- 返回参数
```
{
    
    code: 0|1
    msg: 消息
    data: {
        page_count: 总页数
        list:[
            {
                id: 
                user_id: 回复人
                user_name: 昵称
                face: 用户头像
                news_id: 新闻ID
                content: 回复内容
                follow_id: 0
                notice: {}
                status: 状态
                add_time: 回复时间
            }
            ...
        ]
    }
}
```
